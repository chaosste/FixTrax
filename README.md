## About

FixTrax is a professional AI-powered audio restoration suite designed for vinyl-derived recordings. Using Gemini-driven spectral synthesis, it brings studio-quality restoration to recordings that deserve better than they got ‚Äî suppressing hiss, filtering clicks and pops, and recovering transient energy that the vinyl medium ate.

## Key Features

- **High-Fidelity Hiss Suppression** ‚Äî Intelligent noise reduction that preserves musical detail
- **Click & Pop Filtering** ‚Äî Automatic detection and removal of vinyl surface artifacts
- **Transient Energy Recovery** ‚Äî Restore punch and presence lost in the transfer chain
- **Gemini Spectral Synthesis** ‚Äî AI-driven spectral analysis and reconstruction
- **Preset System** ‚Äî Save and recall restoration configurations
- **Track List Management** ‚Äî Batch processing workflow for albums and collections
- **Multi-Format Export** ‚Äî Output in multiple audio formats
- **A/B Comparison** ‚Äî Instant switching between original and processed audio

## Tech Stack

- **Frontend:** React + TypeScript, Vite
- **AI:** Google Gemini API (spectral synthesis)
- **Audio:** Web Audio API
- **Deployment:** Docker / Google Cloud
